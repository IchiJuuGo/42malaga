### EN LA CONSOLA ###
En la carpeta del proyecto que vaya a entregar:
git init
git add .
git commit

Al subir el commit sin -m te salta el archivo. Hay que agregar
a la linea de arriba:
ADD: all proyects

git remote add 42 <git repository de la intra asociado al proyecto>
git push 42 master

Tambien tenemos tambien para comprobar los procesos:
git status
git log
git remote
